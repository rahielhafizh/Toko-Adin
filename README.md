# Toko-Adin
Pak Adin's Grocery Store is a customized solution aimed at simplifying transaction management at Pak Adin's Grocery Store. Developed using JavaScript and Firebase, the platform records detailed transaction information including buyer name, purchase date, product, and quantity. The project is designed to offer a simple yet effective solution to help manage and analyze transactions more accurately and easily. Advanced evaluation and filtering functions are included, allowing Pak Adin to track and analyze sales data. This website uses Firebase for data storage and visualization tools to improve operational accuracy and decision-making capabilities. The project highlights the proficiency in developing user-friendly interfaces and leveraging cloud infrastructure for business analytics.
